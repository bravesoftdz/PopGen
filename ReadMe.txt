ReadMe zu "| Shorei | PopulationsGenetik"
-----------------------------------------


Dieses Programm simuliert, wie der Name schon sagt, den biologischen Vorgang der Populationsgenetik. Dabei beziehen sich die Berechnungen des Programms auf das "Hardy-Weinberg-Gesetz".

Die Benutzung des Programms ist relativ einfach. Zuerst gibt man an, ob die Selektion gegen das dominante oder das rezessive Gen durchgeführt werden soll. Danach gibt man für das gewählte Gen die Fitness und die Häufigkeit ein, wobei die beiden Werte in der Form von "0,xxx" sein müssen. (für "xxx" geben sie bitte den zu benutzenden Wert ein)

Danach können sie noch die Breite der Ausgabe bestimmen, wobei die Breite "70" als Standard gesetzt ist und die beste Darstellung für eine Auflösung von "800x600" bietet. Sollten Sie das Ergebnis allerdings lieber ausdrucken wollen, empfiehlt es sich, ein Häkchen bei der Box mit der Aufschrift "Ansicht soll gedruckt werden können" zu machen. Dadurch wird ein, für den Druck von Din-A4 - Seiten passender, Wert für die Breite gesetzt.

Nun müssen Sie nur noch auf "Start" drücken, um den Vorgang zu starten.

ACHTUNG: Um den Vorgang wieder zu beenden, MÜSSEN sie auf "Stopp" drücken, da die Simulation sonst unendlich weitergeführt wird.

Nachdem Sie auf "Stopp" gedrückt haben, sehen Sie das Ansichtsfenster mit dem schon fertigen Simulationsergebnis. Durch scrollen der rechten Darstellung können Sie sich nun das gesamte Ergebnis ansehen. Wenn Sie auf eine Zeile in der linken Darstellung drücken, wird in der rechten Darstellung genau die Generation selektiert, die der Zahl der linken Darstellung entspricht. Umgekehrt könne Sie eine Zeile der rechten Darstellung markieren, um zu erfahren, die wievielte Generation dies ist.

Sollten Sie das Ergebnis nun drucken wollen, brauchen Sie im Menü dieses Fensters nur auf "Ansicht" und "Drucken" klicken, um in den Einstellungsdialog für den Drucker zu kommen. Sollten Sie das gesamte Ergebnis drucken wollen, müssen Sie hier auf "Alle" im "Druckbereich" und auf "OK" klicken.

Währe es Ihnen allerdings lieber, nur einen Teil des Ergebnisses zu drucken, so klicken Sie auf "Abbrechen" und unten im Formular auf die Box mit der Aufschrift "automatisch scrollen", um dessen Häkchen zu entfernen. Nun können Sie in der rechten Darstellung die gewünschten Linien durch einfaches anklicken, Maustaste gedrückt halten, mit der Maus nach oben oder unten ziehen und Maustaste loslassen, den zu druckenden Bereich markieren.

Anschließend klicken Sie auf "Ansicht" und "Drucken", wählen unter "Druckbereich" die Auswahl "Markierung" und klicken auf "OK".

Danach erscheint ein Fenster, dass Sie fragt, ob die getätigten Einstellungen für die Simulation mitgedruckt werden sollen. Klicken Sie hier einfach auf "Ja" oder "Nein", um den Druckvorgang zu starten.

Sollten Sie mit dem Drucken fertig sein, können Sie durch einen Klick auf die "x"-Fläche des Formulars das Ansichtsfenster schließen. Sollten Sie es an dieser Stelle noch einmal öffnen wollen, klicken Sie im Hauptformular im Menü einfach auf "Ansicht" und "Ansichtsformular anzeigen".

Sollten Sie eine neue Simulation starten wollen, klicken Sie einfach auf "Datei" und "Neu".
